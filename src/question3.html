Ejercicio 3

Escribe una función en JavaScript que tome un arreglo de números como parámetro y devuelva el número más grande.

function getMax(numbers = []) {
  let max = numbers[0]

  for(let i= 0; i < numbers.length; i++) {
    if(number[i] > max) {
      max = number[i]
    }
  }

  return max
}

La función getMax() toma como parámetro el arreglo vacío numbers, y a continuación dentro de la función, se declara e inicializa la variable max pasándole el valor del primer elemento en el arreglo numbers[0]. max almacenará el valor más grande que se encuentre en el arrreglo.

Con el ciclo for se recorre el arreglo numbers comenzando en 0 (índice del primer elemento) y se avanza mientras i sea menor que la longitud del arreglo numbers.length.

Dentro del ciclo, el elemento actual del arreglo numbers[i] se compara con el valor actual máximo max y se establece una condición, si el elemento actual es mayor que el valor máximo numbers[i] > max significa que se ha encontrado un nuevo valor máximo, entonces, max se actualiza al valor del elemento actual max = number[i].

Al terminar de recorrer todos los números dentro del arreglo, max contiene el valor máximo encontrado en el arreglo numbers y este valor es retornado en la salida de la función.
