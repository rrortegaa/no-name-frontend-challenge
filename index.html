Parte 1: Conceptos teóricos

    HTML/CSS

      Explica la diferencia entre margin y padding en CSS.

      margin es una propiedad usada para referirnos al espacio que se encuentra fuera del borde de un elemento HTML. Nos permite controlar el espacio entre los bordes de un elemento y otros elementos cercanos. Los márgenes pueden ser positivos o negativos y así aumentar o reducir el espacio entre elementos.

      padding por otra parte, es el espacio que se encuentra dentro del borde de un elemento, entre el contenido del elemento y su borde. Controla el espacio entre el contenido del elemento y su propio borde.

      La diferencia entre margin y padding radica en que el margen controla el espacio fuera del borde del elemento HTML y el relleno se encarga del espacio dentro del borde del elemento.

      ¿Qué es el modelo de caja en CSS y cuáles son sus componentes principales?

      El modelo de caja básico de CSS es un estandar que describe como deben representarse los elementos HTML por el motor de renderizado del navegador e indica que cada elemento es considerado como una caja rectangular con cuatro áreas esenciales: área de contenido, área de relleno, borde de la caja y margen de la caja. Cada una de estas áreas tiene un papel específico en la disposición y apariencia del elemento en la página.

      Content es el área central del elemento y contiene el contenido real (texto, imágenes u otros elementos HTML). El tamaño del contenido está determinado por las propiedades width y height del elemento.
